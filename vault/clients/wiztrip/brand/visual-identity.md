# Identidad Visual — WizTrip

> Fuente: Brandbook 2026 (Brandissimo · 17/04/2026)

## Logo

### Concepto

Una marca funcional que equilibra tecnología y cercanía. El logo se apoya en una estructura sans serif de bajo contraste con curvas amplias y terminaciones suaves para que la marca se sienta accesible, humana y cálida, priorizando lectura clara y estable.

La **"W"** es el gran gesto identitario: un trazo continuo y dinámico que representa el recorrido del viaje. En su resolución, suma un punto que funciona como abstracción de un pin de ubicación, conectando con la experiencia del viaje.

Los ajustes tipográficos para equilibrio:
- **Reducción del peso en la "Z"**: evita que el logo se sienta pesado en el centro, lo hace más liviano.
- **Uso de "T" en versalitas**: genera un quiebre con personalidad, le da un toque único sin perder la armonía.

### Variantes y cuándo usar cada una

| Variante | Cuándo usarla |
|---|---|
| **Logotipo** (uso principal) | Cuando la marca tiene contexto. Redes sociales (donde el nombre es protagonista), sobre fotografía, piezas de marca para usuarios que ya conocen WizTrip. |
| **Logotipo + Tagline** ("VIAJÁ CON VENTAJA.") | Campañas y pauta — fundamental para que el nuevo usuario entienda qué hacemos. Material institucional (presentaciones, papelería, firmas de mail). Merchandising. |
| **Isotipo** (solo el ícono) | Espacios reducidos donde el logo completo pierde legibilidad: íconos web, favicons. Como remate visual o detalle decorativo en layouts complejos. Perfiles de RR.SS. o dentro de la plataforma (App/Web). |

### Área de resguardo

La altura de la "W" del logo se toma como unidad **X**. Se define un área de resguardo de **X × X** alrededor del logo. Nada debe invadir ese margen.

### Tamaños mínimos

| Aplicación | Logotipo completo | Isotipo |
|---|---|---|
| Web / digital | 110 px | 35 px |
| Print | 2.9 cm | 1.5 cm |
| Logotipo + Tagline | 1380 px / 3.6 cm | — |

### Usos incorrectos (NO HACER)

- ❌ Colores ajenos a la paleta cromática
- ❌ Combinaciones de colores incorrectas (ver paleta abajo)
- ❌ Logo negro sobre fotografía oscura
- ❌ Estirar (deformar proporciones)
- ❌ Cambiar el avión (pin) de lugar
- ❌ Girar el logo

## Paleta de colores

### Colores primarios (semánticos)

| Color | HEX | Significado |
|---|---|---|
| Magia | `#4C167C` | Violeta profundo. Sabiduría, magia, profundidad de la experiencia. Color anclaje de Wizzo. |
| Wizzo Gold | `#FAB402` | Dorado vibrante. Valor real de las recomendaciones. Color anclaje de Wizzo. |

### Colores de soporte

| Color | HEX | Significado |
|---|---|---|
| Originalidad | `#B99FFF` | Lavanda. |
| Humanidad | `#024384` | Azul mar profundo. |
| Inteligencia | `#E5F53F` o similar verde-amarillo vibrante | Cielo / luz / claridad. |
| Crema | `#FFF2CA` | Fondo cálido para áreas amplias. |
| Negro | `#222524` | Tipografía y elementos de alto contraste. |
| Wizzo Gold transparencia | `#F4F4E9` (apox) | Versión clara/translúcida del oro. |

### Reglas de uso

- **Nunca usar colores ajenos a esta paleta** (regla del logo, aplica a toda la marca).
- En piezas de Wizzo, **siempre** debe estar presente el binomio `#4C167C` (magia) + `#FAB402` (wizzo gold). Es la firma visual del personaje.
- Crema (`#FFF2CA`) para fondos cálidos extensos.
- Negro (`#222524`) para tipografía sobre fondos claros — preferir antes que negro puro `#000000`.
- Sobre fotografía, evitar el uso de logo en negro si la foto es oscura.

## Tipografías

| Función | Tipografía | Características |
|---|---|---|
| **Títulos** | Bricolage Grotesque | Bold, contemporánea, divertida, con personalidad. |
| **Texto corrido** | Host Grotesk | Neutra, limpia, profesional. Sostiene al diseño. |
| **Detalle / editorial** | Noto Natalisq (o similar serif con carácter) | Contraste, textos pequeños, uso editorial. |

### Reglas tipográficas

- **Títulos siempre en Bricolage Bold**. Nunca usar Host Grotesk para títulos principales.
- **Cuerpo de texto en Host Grotesk**. Nunca usar Bricolage para texto corrido (cansa la lectura).
- **Noto Natalisq** se reserva para acentos editoriales — citas, ledes, pull quotes. No es tipografía utilitaria.
- Combinaciones: título Bricolage Bold + cuerpo Host Grotesk Regular + detalle Noto Natalisq Italic funciona bien.

## Reglas operativas para los agentes

1. Cuando un agente genere o describa una pieza visual, **siempre debe nombrar los hex codes** de la paleta. No "color violeta" — "violeta `#4C167C` (magia)".
2. **Tipografías siempre por nombre**: Bricolage para títulos, Host Grotesk para cuerpo, Noto Natalisq para detalles.
3. **Si Wizzo está en una pieza**, la paleta principal son `#4C167C` + `#FAB402`. Esa es la firma de Wizzo.
4. **Cero usos prohibidos.** Si un agente sugiere "logo en negro sobre foto oscura", está mal. Si sugiere estirar el logo, está mal.
5. **Crema (`#FFF2CA`) es el fondo cálido por defecto** cuando no se especifica otro.
